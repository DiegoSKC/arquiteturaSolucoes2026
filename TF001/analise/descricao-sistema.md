# Principais Funcionalidades

- Compra e download de jogos

- Biblioteca pessoal e gerenciamento de atualizações

- Multiplayer 

- Comunidade (chat, grupos, fóruns, reviews, oficina(mods))

- Marketplace (itens virtuais, skins, cartas)

# Tipos de usuários:

- Jogadores casuais

- Jogadores competitivos

- Desenvolvedores (publicam jogos)

- Moderadores e administradores

# Contexto de uso: Desktop (Windows, macOS, Linux), Mobile (app para Android/iOS), Web.
